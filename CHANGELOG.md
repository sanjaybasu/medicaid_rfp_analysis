# changelog

all notable changes to this project will be documented in this file.

the format is based on [keep a changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [semantic versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-06

### added
- complete analysis pipeline for 372,283 accountability claims
- rag+llm claim extraction using claude sonnet 4.5
- per-file normalized concordance analysis
- stratified temporal and regional analyses
- manuscript figures (3 figures: bar chart, temporal trends, heat map)
- data verification scripts
- complete documentation and reproducibility materials
- harvard dataverse deposit (doi: 10.7910/DVN/6EFL00)

### changed
- reframed using principal-agent theory
- normalized concordance by file count
- stratified analyses by document type

### fixed
- document count clarification (265 source → 1,668 processed)
- temporal patterns attribution to document type

## [unreleased]

### planned
- additional sensitivity analyses
