# data

## complete dataset

all analysis data is available at harvard dataverse:

**[https://doi.org/10.7910/DVN/6EFL00](https://doi.org/10.7910/DVN/6EFL00)**


## source documents

original procurement documents (265 PDFs and ZIP archives, ~8 GB) are not included in this repository due to size constraints.

researchers can obtain source documents through:

1. **direct download** - 78% publicly available on state procurement websites (urls in document_inventory.csv)
2. **procurement portals** - 18% require registration on state e-procurement systems
3. **foia requests** - 4% obtained through freedom of information act requests

## reproducibility

the complete extracted claims dataset at dataverse enables replication of all analyses without requiring source document access.
